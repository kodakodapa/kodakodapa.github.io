MARKDOWN QUICK REFERENCE
========================

HEADINGS
--------

## H2 - Section heading
### H3 - Subsection
#### H4 - Smaller heading
##### H5 - Even smaller
###### H6 - Smallest heading

---------------
**bold text**
*italic text*
***bold and italic***

-----
Unordered:
- Item one

  - Another nested
- Item three

Ordered:
1. First item
2. Second item
   1. Nested numbered
   2. Another nested
3. Third item


```
Plain code block
No highlighting
```
